📌 Overview

The Dialysis Emergency System is a lightweight web application designed to provide instant access to critical patient data during medical emergencies.

It helps doctors, hospitals, and family members quickly retrieve essential dialysis-related information — saving time and potentially lives.

✨ Key Features
⚡ Instant patient data access
🌐 Multi-language support
📱 Mobile-friendly UI
🔄 Real-time data from Google Sheets
🧾 Data entry via HTML form page
🧠 Clean and simple interface
🚑 Emergency-ready design

🔄 How It Works
Patient or admin fills data using HTML form page
Data is stored in Google Sheets
System fetches data via OpenSheet API
UI displays patient information instantly
